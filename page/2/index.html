sass基础语法

搭建环境Sass是基于ruby开发的,但是用的人却不用懂得写Ruby. 首先,在编写Sass前要先下载安装Ruby (在网上搜索Ruby的时候 看到的评价都在说这门语言很优雅,以后有空的话会去接触下的!) 👇 然后在命令行中 输入 gem install sass 等待执行完毕后 然后输入在命令行输入 sass -v 如果出现了相应的版本号 就意味着安装成功了👏 开始基础语法注释/*注释*/ 👈这种注释会在编译后的CSS中显示 //注释 👈这种则不会 变量$变量名字:变量属性值 使用! default 会让之前的样式覆盖掉当前的样式 //你好 我是sass $defaultColor: red; $defaultColor: #0000ff !default; h1{color:$defaultColor;} 编译后的css: h1 { color: red; } 关于变量的作用域💡需要注意的是如果上述变量的作用域在所有类的外面,那它就是全局变量,否则他的作用域只在当前的类 嵌套CSS规则/* SASS嵌套 */ $defaultColor: red; #c
