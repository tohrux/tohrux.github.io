搭建环境

Sass是基于ruby开发的,但是用的人却不用懂得写Ruby.

所以编写Sass前要先下载安装Ruby

(在网上搜索Ruby的时候 看到的评价都在说这门语言很优雅,以后有空的话会去接触下的!)

👇

然后在命令行中 输入 

gem install sass

就可以安装了

开始基础语法

变量

$变量名字:变量属性值

使用! default 会让之前的样式覆盖掉当前的样式

//你好 我是sass

$defaultColor: red;

$defaultColor: #0000ff !default;

h1{color:$defaultColor;}

输出的css:

h1 {
  color: red;
}

关于变量的作用域

💡需要注意的是如果上述变量的作用域在所有类的外面,那它就是全局变量,否则他的作用域只在当前的类

嵌套CSS规则

/* SASS嵌套 */
$defaultColor: red;
#content{
    article{
        h1{color: $defaultColor;}
        p{color: $defaultColor;}
    }
    aside{
        color: aqua; 
    }
}

输出的css:

#content article h1 {
  color: red;
}

#content article p {
  color: red;
}

#content aside {
  color: aqua;
}
